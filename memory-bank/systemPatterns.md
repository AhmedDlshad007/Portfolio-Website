# System Patterns

## System Architecture
- Built with Next.js framework
- React for component-based UI
- TypeScript for type safety
- CSS for styling
- Azure AI integration for chatbot functionality

## Key Technical Decisions
- Using Next.js for server-side rendering and improved SEO
- Client-side state management with React hooks
- TypeScript for improved code quality and developer experience
- Integration with Azure AI for chatbot capabilities

## Design Patterns
- Functional components with hooks for state management
- Mobile-first responsive design
- Component composition for UI elements

## Component Relationships
- Layout component for overall page structure
- Home component containing page sections
- Chatbot component for AI interaction
- Navigation components for site navigation
