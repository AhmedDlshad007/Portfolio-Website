# Portfolio Website with OpenAI Chatbot

This is my personal portfolio website showcasing my skills, projects, and experience. The website features an interactive OpenAI-powered chatbot that can answer questions about my resume and skills.

## Live Demo

Visit the live website: [https://ahmed-dlshad-site.netlify.app](https://ahmed-dlshad-site.netlify.app)

## Features

- Responsive design that works on desktop and mobile devices
- Interactive AI chatbot powered by OpenAI GPT-3.5 Turbo
- Sections for skills, projects, and work experience
- Resume download option
- Contact information

## Technologies Used

- Next.js 15.2.4
- React 19.0.0
- TypeScript
- Tailwind CSS
- OpenAI API
- Netlify for hosting

## Getting Started

1. Clone this repository
2. Install dependencies with `npm install`
3. Create a `.env.local` file with your OpenAI API key:
   ```
   OPENAI_API_KEY=your_openai_api_key
   NEXT_PUBLIC_OPENAI_API_KEY=your_openai_api_key
   ```
4. Run the development server with `npm run dev`
5. Open [http://localhost:3000](http://localhost:3000) in your browser

## Deployment

This site is configured for static export to Netlify:

1. Build the project with `npm run build`
2. Deploy to Netlify with `netlify deploy --prod`
